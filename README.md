# LetsKick

Let's Kick Messenger Bot

## How use run it
  - npm install
  - pip install -r requirements.txt

### Functionalities
  - Clubs and players lookup
  - Reminder before matches
  - Live score update
  
### Stacks used
  - Nodejs
